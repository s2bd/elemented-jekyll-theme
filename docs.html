Installation

git clone https://github.com/your-username/elemented-jekyll-theme.git

# Navigate to the directory

# Install dependencies
bundle install

# Start the development server

üß© Using Components

Elemented uses Jekyll's include system to make components reusable. Here's how to use them:

Basic Syntax

{% raw %}{% include component-name.html parameter="value" %}{% endraw %}

Hero Section

Create impressive hero sections with customizable content and buttons.

{% raw %}{% include hero.html 
   title="Welcome to My Site"
   subtitle="Build amazing websites with ease"
   size="large"
   background_image="/assets/images/hero-bg.jpg"
   buttons=page.hero_buttons
%}{% endraw %}

Parameters:

  • title - Main heading text
  • subtitle - Subheading text
  • size - "normal" or "large"
  • background_image - Path to background image
  • buttons - Array of button objects

üé® Customization

CSS Variables

Customize colors and spacing using CSS variables:

:root {
  --primary-color: #3b82f6;
  --secondary-color: #1e40af;
  --accent-color: #f59e0b;
  --text-color: #1f2937;
  --background-color: #ffffff;
}

SCSS Customization

Override default styles by editing _sass/main.scss:

// Custom button styles
.btn {
  border-radius: 2rem;
  font-weight: 700;
  
  &.btn-custom {
    background: linear-gradient(45deg, #ff6b6b, #4ecdc4);
    color: white;
  }
}

Creating Custom Components

Create your own components in the _includes directory:

<!-- _includes/custom-component.html -->
<div class="custom-component">
    <h3>{{ include.title }}</h3>
    <p>{{ include.description }}</p>
    {% raw %}{% if include.button_text %}
    <a href="{{ include.button_url }}" class="btn">{{ include.button_text }}</a>
    {% endif %}{% endraw %}
</div>

‚ùì FAQ

How do I add a new component?

Create a new file in the _includes directory and use Jekyll's include syntax to add it to your pages.

Can I customize the colors?

Yes! Edit the CSS variables in _sass/main.scss or override them in your custom CSS.

How do I add custom JavaScript?

Add your JavaScript files to the assets/js directory and include them in your layout files.

Is the theme mobile responsive?

Yes! All components are built with a mobile-first approach and work perfectly on all devices.
